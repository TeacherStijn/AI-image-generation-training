# Image generation

This is an example of a prompt library. You can create headings using hashtags.
The advantage of an .MD file is that it is easy to generate or read using tooling.

Below is an example of a way to categorize:


## Corporate identities

### Corporate identitie of CompanyX

#### Prompt
`Modern minimalistic corporate identity for CompanyX, clean typography, strong geometric logo mark, professional branding presentation on stationery mockups, white and deep blue color palette, high-end brand manual layout, ultra sharp details --ar 1:1 --q 1 --c 10`

### Imitating a corporate identity style

#### Prompt
`Corporate branding presentation in the style of a premium Scandinavian tech company, minimal layout, neutral tones with subtle accent color, grid-based design system, logo variations, business cards and website mockups, soft studio lighting --ar 1:1 --q 1 --c 15`
