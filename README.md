# Documentation for the AI image generation course

## Contents
In this repository you will find:
- Exercises you can do during the AI image generation course
- Demo's that [Stijn](https://www.linkedin.com/in/stijnjanssen83/) has used during his course to transfer knowledge ;) 
- Tooling (under construction) that can help you more easily integrate A.I. within your processes

Please check the disclaimer below before using this content.

- All prompts shared here can be freely used to generate your own images
- When using or adapting any tools shared here, please share your experiences on [LinkedIn](https://www.linkedin.com/)
